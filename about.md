# What is he doing?

Show other people what you're doing by automatically sending webhook messages upon big actions such as playing a level or opening up the editor!

Will be updated to add more actions/scene changes/configurations in the future :3

## Currently logged actions

Each of these can be turned on or off in the mod settings under **Notification Toggles**.

- **Opening / exiting Geometry Dash** (with session time)
- **Playing / exiting a level** (with session info and run time)
- **Editor playtest** (NOT in the editor)
- **Level complete** (with session info and run time)
- **New best** (when you finally get a new best on the level)
- **Opening / exiting the editor** (when you enter or leave the level editor)
- **Idle** (after you have been idle for 5 minutes, 30 minutes, or 2 hours.)
